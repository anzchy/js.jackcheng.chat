## arrow functions

arrow functions in JavaScript is similar to lambda expressions in Python.
arrow function is usually used in combination with map/filter/reduce methods.

### map method

<PythonEditor title="arrow function vs lambda expressions" compare={true} height={150}>

```javascript !! js
const nums = [1, 2, 3, 4];
const squares = nums.map(x => x * x);
console.log(squares);
```

```python !! py
# similar expression in python
nums = [1, 2, 3, 4]
squares = list(map(lambda x: x**2, nums))
print(squares)
```
</PythonEditor>

### filter method

<PythonEditor title="filter method" compare={true} height={150}>

```javascript !! js
// arrow function + filter method
const nums = [1, 2, 3, 4];
const evens = nums.filter(x => x % 2 === 0);
console.log(evens);
```

```python !! py
# lambda expression + filter method
nums = [1, 2, 3, 4]
evens = list(filter(lambda x: x % 2 == 0, nums))
print(evens)

```
</PythonEditor>
